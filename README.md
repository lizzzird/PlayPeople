# PlayPeople
Игровой журнал представляет собой веб-сайт, содержащий список игр, ленту новостей и календарь релизов.
## Правила коммуникации в команде
- для связи используем телеграм
- регулярно обсуждаем прогресс, задачи и возникающие проблемы
- каждый член команды имеет свои обязанности и роли
- ведём документацию проекта
- поддерживаем добрую атмосферу
## Стиль программного кода
- C# ASP.NET
- следуем стандарту кодирования Microsoft:
  - https://learn.microsoft.com/en-us/dotnet/csharp/fundamentals/coding-style/identifier-names
  - https://learn.microsoft.com/en-us/dotnet/csharp/fundamentals/coding-style/coding-conventions
- добавляем комментарии к неочевидным частям кода
- документируем публичные методы и классы с использованием XML-комментариев
- используем Entity Framework для работы с базой данных
## Алгоритм действий
- склонируйте репозиторий на свой компьютер
- начните работать над проектом
- когда вы готовы сохранить свои изменения, добавьте их в индекс, затем сделайте коммит, введя сообщение о коммите, описывающее изменения, затем отправьте свои изменения на гитхаб
